# Retail Sales Analytics Case Study
## Overview

Performed end-to-end Exploratory Data Analysis on 1M+ retail transactions to uncover revenue drivers, customer behavior patterns, and retention dynamics.
Data Source : [Online Retail II UCI](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)

This project demonstrates business-focused analytics using Python.

## Key Insights

Revenue is highly concentrated among top customers (Top segment drives ~51%).

Strong Q4 seasonal spike driven by holiday demand.

Customer retention drops sharply after first purchase.

Returns account for ~2.15% of transactions but impact ~7% of revenue.

## Advanced Analytics Performed

Revenue & distribution analysis (log-transformed)

Seasonal trend analysis

Spend-based customer segmentation

RFM behavioral segmentation

Cohort retention modeling

Returns impact analysis

## Tools Used

Python • pandas • numpy • matplotlib • seaborn • Jupyter Notebook
